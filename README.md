#Upthere Camera Mockup

### Installation

- clone this repo:
    - "git clone https://github.com/jamesmcintyre/mockup-technical.git"
- cd into cloned repo:
    - "cd mockup-technical"
- open a terminal and run:
    - "npm i"
- then run:
    - "npm start"

- you could also run "http-server" in the "/public" folder if you have http-server installed globally
